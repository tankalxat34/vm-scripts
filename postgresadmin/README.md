Здесь находятся скрипты автоматизации DBA PostgreSQL.

Для установки и настройки кластера PostgreSQL порядок вызова скриптов:
1. `create_postgres.sh`
2. `compile_postgresql.sh`
3. `install_postgresql.sh`

Для удаления кластера PostgreSQL необходимо выполнить скрипты в порядке:
1. `remove_postgresql.sh`
2. `remove_postgres.sh`
